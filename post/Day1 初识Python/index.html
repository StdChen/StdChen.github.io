Day1 初识Python

2020-05-17 5 min read # Python

1 环境搭建

推荐使用Anaconda与PyCharm作为Python解释器环境和IDE。Anaconda是一个集成包，集成了数据分析等几乎所有的常用安装包，如numpy，scipy，matplotlib等。不建议使用官方原始Python自己一个一个安装所有的包。搭建完成后，以Anconda下的Python作为默认解释器，PyCharm作为开发环境，相当于编辑器。下面介绍windows环境下Python环境的搭建。

  • 从官网下载Anaconda与PyCharm-community，由于Anaconda官网下载较慢，可以直接从清华大学镜像站下载。
  • 安装Anaconda，路径中不能包含空格。
  • 安装PyCharm，安装选项中选择创建桌面快捷方式即可，其余选项均默认。安装完成后打开进行PyCharm的配置，直接选择下一步。然后选择Creat New Progect，Location选择自己的Python test文件夹（例如，E:\TEST\Python test），对于interperter选项，PyCharm会默认识别计算机已安装的解释器。
  • Windows下设置PyCharm在Anaconda的编译环境。打开PyCharm，点击File->settings->Project: Python test，选择Project Interpreter，点击右侧设置按钮，选择Add，然后选择Conda Environment，在Exising enviornment中浏览文件，根据Anaconda的安装路径（例如，D:\ProgramData\Anaconda3\Tools\python.exe）找到Python编译环境，选择Apply即可。至此，Python环境搭建完成。

2 Python代码中的注释

  • 单行注释：以#和空格开头，用于注释单行代码。
  • 多行注释：以"""或'''开头，以"""或'''结尾。
'''
This is my first python program - Hello, world!
Version: 0.1
Author: StdChen
'''
print('Hello, wordl!')
# print("你好，世界！")
